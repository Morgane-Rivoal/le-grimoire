# Assets du Grimoire

- `illustrations/` : planches botaniques SVG utilisées dans Explorer et les fiches.
- `images/` : futures images fixes de l’interface.
- `icons/` : futures icônes de l’application.
- `photos/` : emplacement prévu pour de futures photos locales si l’application devient installable/mobile.

Règle de direction artistique : les plantes affichées dans Explorer doivent utiliser une planche botanique, jamais une photo brute. Les photos d’identification restent dans la fiche comme preuve d’observation.
